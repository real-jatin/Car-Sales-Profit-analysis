# 🚗 Car Sales & Profit Analysis Dashboard

📌 Short Description / Purpose

The Car Sales Dashboard is an interactive Excel-based report designed to analyze car sales, profit margins, pricing differences, and regional performance. It helps identify profitable segments, loss-making models, and state-wise sales trends to support smarter business decisions.

🛠️ Tech Stack

The dashboard was built using:

📊 Microsoft Excel – Main visualization tool
📈 Pivot Tables & Pivot Charts – Data summarization and insights
🔄 Power Query – Data cleaning and transformation
🧠 Excel Formulas – Profit Margin %, Price Difference, KPIs
🧩 Data Modeling – Relationships between brands, models, and regions
📁 File Format – .xlsx

📂 Data Source

Car sales dataset containing:

Sales & profit data
Car brands and models
Selling price and price differences
Usage categories (High / Low usage)
State-wise sales data
🌟 Features & Highlights
🔴 Business Problem

Car dealerships often struggle to:

Identify which brands and models are profitable
Track loss-making units
Understand regional sales performance
Analyze pricing differences across brands
Compare usage-based profitability
🎯 Goal of the Dashboard

To build a centralized Excel dashboard that:

Tracks total sales and overall profit/loss
Identifies high vs low usage profitability
Analyzes brand-wise price differences
Highlights loss-making models
Monitors state-wise sales performance
